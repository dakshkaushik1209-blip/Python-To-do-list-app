# 📝 Python To-Do List

A simple command-line To-Do List application built using Python. This project allows users to manage daily tasks by adding, viewing, marking tasks as completed, and deleting tasks.

It is a beginner-friendly project that demonstrates the use of Python fundamentals such as lists, dictionaries, loops, functions, exception handling, and input validation.

---

##  Features

-  Add new tasks
-  View all tasks
-  Mark tasks as completed
-  Delete tasks
-  Input validation
-  Exception handling for invalid inputs
-  Menu-driven interface

---

## 🛠️ Technologies Used

- Python 3
